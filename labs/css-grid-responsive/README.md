# Mini Lab: Responsive Grid

## 🎯 Goal

This lab will help you:

- Build a **complete layout** with header, sidebar, content, and footer using CSS Grid.
- Create a **responsive card gallery** using `auto-fit` and `minmax`.
- Apply a **professional Grid + Flexbox architecture** for scalable layouts.

---

## 🧩 Lab Structure

.layout
├─ header
├─ aside (sidebar)
├─ main
│   └─ gallery
│       └─ card
└─ footer

---

## ⚡ Key Concepts Applied

- Two-dimensional Grid: rows + columns  
- `grid-template-columns / rows`  
- Fraction units (`fr`)  
- Column spanning: `grid-column: start / end`  
- Gap: spacing between cells  
- Responsive patterns: `repeat(auto-fit, minmax())`  
- Flexbox inside components for internal alignment

## 💡 Pro Tips

- Grid for layout, Flexbox for alignment  
- Use `repeat(auto-fit, minmax())` for responsive designs  
- Keep CSS modular and well-documented
