# Accessibility Foundations

Web accessibility ensures that websites and applications can be used by people with a wide range of abilities and disabilities.

Accessibility is a fundamental aspect of web engineering and is supported by the design of the web platform itself.

## Core Concepts

Accessibility on the web relies on several principles:

- Semantic HTML
- Keyboard navigation
- Screen reader compatibility
- Proper use of landmarks and headings
- Sufficient color contrast

## WCAG

The Web Content Accessibility Guidelines (WCAG) define standards for accessible web content.

The guidelines are based on four principles:

- Perceivable
- Operable
- Understandable
- Robust

## Why Accessibility Matters

Accessible interfaces improve usability for everyone and ensure that digital products can be used by people with visual, motor, cognitive, or auditory impairments.

Accessibility is not an optional feature but a core responsibility of modern web development.
