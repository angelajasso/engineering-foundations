# Flexbox Foundations

Flexbox is a one-dimensional layout system used to distribute space and align elements within a container.

It works along two axes:

- Main Axis
- Cross Axis

A Flex Container is created using:

display: flex;

All direct children become Flex Items.

Core properties used in Flexbox layouts:

- flex-direction
- justify-content
- align-items
- flex

Flexbox is widely used for:

- Navigation bars
- Component layouts
- Centering elements
- Distributing space between items

Understanding the Flexbox mental model allows developers to create predictable and scalable layouts.
