# CSS Foundations

## CSS is a rule system

CSS is not decoration.

It is a conflict resolution system where multiple rules compete to style elements in the DOM.

The browser resolves conflicts using:

1. Cascade
2. Specificity
3. Inheritance
4. Source order

A solid understanding of these mechanisms allows developers to write predictable, scalable, and maintainable styles.

## Engineering principle

Good CSS avoids conflicts instead of fighting them.
