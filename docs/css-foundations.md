# CSS Foundations

## CSS is a Rule System

CSS is not just decoration.

It is a rule-based system where multiple style declarations compete to control the visual presentation of DOM elements.

The browser resolves these conflicts using four mechanisms:

1. Cascade
2. Specificity
3. Inheritance
4. Source Order

Understanding these mechanisms allows developers to write predictable and maintainable stylesheets.

Without this knowledge, CSS often appears to work "by accident".
