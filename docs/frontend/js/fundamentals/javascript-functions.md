# Functions in JavaScript

## Definition

A function is a reusable block of code that receives data, executes logic, and returns a result.

---

## Types of Functions

- Function declarations
- Function expressions
- Arrow functions

---

## Parameters and Arguments

- Parameters: variables defined in the function
- Arguments: actual values passed when invoking the function

---

## Return

Functions can return values using `return`.  
If not specified, they return `undefined`.

---

## Single Responsibility Principle

A function should have only one responsibility.

### Benefits:
- Clearer code
- Easier maintenance
- Reusability

---

## Pure Functions

Functions that:
- Do not depend on external state
- Do not produce side effects
- Always return the same result

---

## Best Practices

- Use descriptive names
- Avoid long functions
- Separate responsibilities
